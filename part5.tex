Настоящая глава посвящена спектральному анализу 296 коротких всплесков Конус-Винд,
локализации которых была описана в главе~\ref{chapt2}.

Наблюдаемые спектры гамма-всплесков хорошо описываются моделью Банда (Band)~\citep{Band_1993ApJ}
\begin{equation}\label{eq:Band_2}
\frac{dN}{dE}=A \left\{
\begin{array}{lr}
\left(\frac{E}{E_\rmn{n}}\right)^\alpha \exp\left(-\frac{E}{E_0}\right) \mbox{, } 
&\mbox{если } E<(\alpha-\beta)E_0\\
\left(\frac{E}{E_n}\right)^\beta 
\left[(\alpha-\beta)\left(\frac{E_0}{E_\rmn{n}}\right)\right]^{(\alpha-\beta)} 
\exp(\beta-\alpha)  \mbox{, } &\mbox{если } E\geq(\alpha-\beta)E_0 \\
\end{array}
\right. \mbox{ ,}
\end{equation}
здесь $\alpha$~--- показатель степени в области малых энергий, 
$\beta$~--- показатель степени в области больших энергий. Для ярких всплесков по данным BATSE 
и \textit{Fermi}-GBM эти параметры имеют значения имеют значения $\alpha \sim -1^{+1}_{-1}$ и 
$\beta \sim -2.5^{+1}_{-2}$~\citep{Goldstein_2013ApJS, Gruber_2014ApJS}. 
Энергия, на которую приходится максимум в спектре $E F_E = E^2 dN/dE$ равна $E_\rmn{p}=(\alpha+2) E_0$.

В некоторых случаях спектр можно описать степенной моделью с экспоненциальным завалом
\begin{equation}\label{eq:CPL_2}
\frac{dN}{dE} = A \left(\frac{E}{E_\rmn{n}}\right)^\alpha \exp\left(-\frac{E}{E_0}\right) \mbox{ ,}
\end{equation}
это по сути предел модели Банда при $\beta \to -\inf$.

Для слабых всплесков, когда излом спектра невозможно определить из-за малого числа отсчётов 
в жесткой части спектра, используется степенная модель
\begin{equation}\label{eq:CPL_2}
\frac{dN}{dE} = A \left(\frac{E}{E_{\rmn{n}}}\right)^{\alpha}
\end{equation}

До сих пор нет единого мнения о физическом механизме генерации $\gamma$-квантов. 
Наиболее распространённая модель~--- генерация  $\gamma$-квантов во внутренних 
ударных волнах струйного выброса. В соответствии с этой моделью 
часть кенетической энергии потока преобразуется в энергию ускоренных частиц при столкновении быстро 
движущийся и медленно движущийся испущенной ранее компоненты потока. Энергия затем излучается частицами 
в виде гамма-квантов посредствам нескольких процессов таких как синхротронное излучение 
и обратное Комптоновское рассеяние. Предпологаемые связи физических параметров и параметров 
описанных спектральных моделей обсуждаются в обзоре~\citep{Kumar_and_Zhang_2014PhR}
