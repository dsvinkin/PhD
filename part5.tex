\chapter{Спектральный анализ коротких всплесков Конус-Винд} \label{chapt5}

Гамма-всплески имеют нетепловой спектр, обычно степень с изломом, максимум которого ($\nu F_{\nu}$) 
лежит в области 100--1000~кэВ.

Из наблюдений изломов во временных профилях оптического и рентгеновского послесвечений 
известно, что излчучение всплесков узконаправлено~--- представляет собой 
струйный выброс (\textit{jet}) с углом раскрытия $\sim 2$--$10^{\circ}$. 
Этот факт был предсказан до первых наблюдений изломов в работе~\citep{Rhoads_1999ApJ}. 
С учётом коллимации энерговыделение гамма-всплесков лежит в диапазоне $10^{42}$--$10^{52}$~эрг.

Еще до полного подтверждения космологической природы гамма-всплесков предполагалось, 
что в их источниках происходит ультрарелятивисткое движение~\citep{Paczynski_1986ApJ, Goodman_1986ApJ, Fenimore_1996ApJ}. 
%(Paczynski, 1986; Goodman, 1986; Fenimore et al., 1996; Piran, 1999)
Подтверждение этой гипотезы было сделано для GRB~970508 по наблюдению радио 
сцинстилляций~\citep{Goodman1997_NewA, Frail_1997Natur} и по наблюдению сверхсветового 
движения для GRB~030329~\citep{Taylor_2004ApJ}, где ударная волна была релятивистской 
спустя несколько недель после взрыва.

До сих пор нет единого мнения о физическом механизме генерации $\gamma$-квантов. 
Наиболее распространённая модель~--- генерация  $\gamma$-квантов во внутренних 
ударных волнах струйного выброса. В соответствии с этой моделью 
часть кенетической энергии потока преобразуется в тепловую при столкновении быстро 
движущийся и медленно движущийся испущенной ранее компоненты потока. Выделившаяся 
тепловая энергия затем излучается в виде гамма-квантов посредствам нескольких процессов 
таких как синхротронное излучение и обратное Комптоновское рассеяние.

Наблюдаемые спектры гамма-всплесков описываются моделью Банда (Band)~\citep{Band_1993ApJ}
\begin{equation}\label{eq:Band_2}
\frac{dN}{dE}=A \left\{
\begin{array}{lr}
\left(\frac{E}{E_n}\right)^\alpha \exp\left(-\frac{E}{E_0}\right) \mbox{, } 
&\mbox{если } E<(\alpha-\beta)E_0\\
\left(\frac{E}{E_n}\right)^\beta 
\left[(\alpha-\beta)\left(\frac{E_0}{E_n}\right)\right]^{(\alpha-\beta)} 
\exp(\beta-\alpha)  \mbox{, } &\mbox{если } E\geq(\alpha-\beta)E_0 \\
\end{array}
\right. \mbox{ ,}
\end{equation}
здесь $\alpha$~--- показатель степени в области малых энергий, 
$\beta$~--- показатель степени в области больших энергий. Для ярких всплесков по данным BATSE 
и \textit{Fermi}-GBM эти параметры имеют значения имеют значения $\alpha \sim -1^{+1}_{-1}$ и 
$\beta \sim -2.5^{+1}_{-2}$~\citep{Goldstein_2013ApJS, Gruber_2014ApJS}. 
Энергия, на которую приходится максимум в спектре $E F_E = E^2 dN/dE$ равна $E_\rmn{p}=(\alpha+2) E_0$.
В некоторых случаях, в основном для слабых всплесков, спектр можно описать 
степенной моделью с экспоненциальным завалом
\begin{equation}\label{eq:CPL_2}
\frac{dN}{dE} = A \left(\frac{E}{E_n}\right)^\alpha \exp\left(-\frac{E}{E_0}\right) \mbox{ ,}
\end{equation}
это по сути предел модели Банда при $\beta \to -\inf$.
